# Brand assets

These images are for submission to the
[home-assistant/brands](https://github.com/home-assistant/brands) repository,
which HACS and Home Assistant use to display integration logos/icons.

They do **not** belong in the integration itself — open a PR to the brands repo
that adds:

```
custom_integrations/lego_power/icon.png      (256x256)  -> brands/icon.png
custom_integrations/lego_power/icon@2x.png   (512x512)  -> brands/icon@2x.png
custom_integrations/lego_power/logo.png                 -> brands/logo.png
custom_integrations/lego_power/logo@2x.png             -> brands/logo@2x.png
```

`icon.png` must be a square PNG (256×256) with a transparent background.
Replace these placeholders with final artwork if you have it.
